# Portfolio Romain Rubens

Portfolio multilingue pour Romain Rubens, étudiant en troisième année de design industriel spécialisé en UX/UI et design d'interaction.

## Langues supportées

- Français (FR)
- English (EN)
- Հայերեն (ՀԱՅ) - Arménien

## Démarrer le projet

```bash
npm install
npm run dev
```

Le site sera accessible sur [http://localhost:5000](http://localhost:5000)

## Technologies

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Radix UI

## Structure du projet

- `/src/app` - Pages et routes de l'application
- `/src/components` - Composants réutilisables
- `/src/components/sections` - Sections principales du portfolio
- `/public` - Assets statiques

## Déploiement

Le projet est configuré pour le déploiement sur Replit avec autoscaling.
